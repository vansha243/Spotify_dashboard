**Spotify Music Analytics Dashboard**

This interactive dashboard provides insights into global music streaming trends, analyzing track performance, audio features, and artist popularity. It helps identify patterns in user listening behavior and highlights top-performing songs and artists over time.


1. Purpose

To enable data-driven decision-making by uncovering trends in music consumption, track characteristics (danceability, energy, valence), and yearly streaming distribution.

2. Tech Stack

Power BI – Data visualization & dashboard development
SQL – Data extraction and transformation
Excel / CSV – Data preprocessing
DAX – Measures, KPIs, and calculated insights

3. Data Source
Dataset sourced from Kaggle (Spotify tracks & audio features dataset)

4. Key Features / Highlights

* KPI cards showing total streams and audio feature distributions
* Interactive slicers (year, artist, key, mode) for dynamic filtering
* Track-level analysis (top streamed songs like *Blinding Lights*)
* Correlation visuals comparing audio features (danceability, speechiness, acousticness)
* Year-wise trend analysis of music releases and streams
* Artist popularity breakdown and key distribution insights
